# vue-crud
## Serve setup
```
npm install -g json-server
```
## Start JSON Server by cmd
```
json-server --watch data/db.json
```
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
